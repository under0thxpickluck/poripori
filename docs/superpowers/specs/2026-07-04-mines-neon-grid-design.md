# Mines ネオングリッド・リデザイン 設計仕様書

作成: 2026-07-04 / ステータス: **実装完了（2026-07-04）**

> 実装結果: vitest 74件全合格 / build 成功 / デモモードE2E（開始→開示→Cash Out→バースト→
> リロード復元）合格。360px/390px×ダーク/ライトのスクリーンショット目視確認済み。

## 1. 目的

Mines（旧・宝石堀り）のビジュアルと効果音を「最先端・サイバー」＝**ネオングリッド**
（サイバーパンク直球）方向に全面刷新する。候補4案からユーザーが選定済み。
世界観は宝石堀りから**「電脳グリッドのデータ採掘」**へ読み替える。

## 2. スコープ

### やること
- `/mines` のゲーム体験全部: Board / Tile / HUD / HistoryCards / CelebrationOverlay / Mines.tsx
- 効果音（`sounds.ts`）のシンセ化（Web Audio 合成のまま波形・エンベロープ変更）
- ホーム `GameCorner.tsx` の Mines カード（配色・文言）

### やらないこと（不変）
- ロジック全部: `useMinesGame.ts` / RPC / `mines-math.ts` / DB（migrate-015）
- タイルの `aria-label`（"マス N"。E2E とアクセシビリティの互換維持）
- `/admin/mines`、Plinko、`haptics.ts`
- 外部フォント・画像・ライブラリの追加（CSS + lucide + framer-motion で完結。バンドル増ゼロ）

## 3. ビジュアル言語

| 要素 | 値 |
|---|---|
| ベース背景 | `#0a0a12`（漆黒） / パネル `#10101c` |
| プライマリ | シアン `#00f0ff`（枠・発光・勝ち） |
| セカンダリ | マゼンタ `#ff2ec4`（アクセント・Cash Out グラデ） |
| 危険 | 赤 `#ff2e2e` 系 + グリッチ |
| 数値 | 等幅（`ui-monospace` スタック、`tabular-nums`）でテレメトリ風 |

- ステージ背景に微細なグリッド線（CSS `repeating-linear-gradient`、画像なし）と
  ゆっくり流れる走査線（scanline）。
- 用語: 「罠」→**「トラップ」**に統一（HUD・ページコピー・title 属性）。
- コピー例: ヘッダー「グリッドに潜むトラップを避けて、データを採掘せよ」。

## 4. コンポーネント別仕様

### Tile.tsx
- 未開: 暗パネル + 低輝度シアン1px枠。ホバーで枠輝度↑ + 微グロー。
- 安全開示: 宝石 → **ホロクリスタル**。ネオン4色バリエーション
  （シアン/マゼンタ/ライム/イエロー。`gemKind` の疑似ランダム決定は流用）。
  開示時にネオンリングが拡散（scale+fade のリング1個。軽量）。
- 罠開示: **グリッチ**＝数フレームの RGB ずれシェイク（translate ±2px + 色ずれ影）
  + 赤スキャンライン。アイコンはドクロ（Skull）→ ウイルス（`Bug`）。
- `locked` / `pending` の状態区分は現行踏襲。

### Board.tsx
- コンテナ: グリッド線発光の「電脳空間」。タイル間ギャップにシアン微光。
- スタッガー出現は現行の delay 計算を流用。

### HUD.tsx
- テレメトリパネル: 四隅コーナーブラケット装飾（擬似要素 or 小 span）。
- 倍率: 等幅フォント + framer-motion のデジタルロール（現行の spring 差し替え）。
- Cash Out: 金 → **シアン→マゼンタのグラデ** + 外周グロー。ホバーでスキャン光が走る。
- 開始ボタン: シアン系。ラベル「掘りはじめる」→「採掘開始」。

### HistoryCards.tsx
- 勝ち: シアンネオン枠 / 負け: 赤グリッチ（絵文字 💥 → ⚠ 系表示）。
- 統計チップ（勝率/最高/連勝）もネオン枠に。

### CelebrationOverlay.tsx
- cashout: シアン系グロー + 獲得額デジタルカウントアップ（framer-motion の
  `animate` で数値補間、`useReducedMotion` 時は即値表示）。
- 20x 以上: 画面全体のグリッドフラッシュ（radial → grid 発光）。

### Mines.tsx
- ヘッダーアイコン: Gem → `Cpu` or `Grid` 系。説明文をサイバー文言に。
- フッター注記の「罠」表記を「トラップ」へ。

### GameCorner.tsx（Mines カードのみ）
- 配色: emerald → シアン/マゼンタ。文言「グリッドに潜むトラップを避けろ」
  「Mines 🕹 開けるほど倍率アップ、引き際はキミ次第」（desc の倍率文言は維持可）。

### sounds.ts（波形のみ変更、API 不変）
- click: 矩形波の短ブリップ / gem: シンセアルペジオ（開示数で音程上昇は維持）
- multiplier: FM 風ピコ / cashout: 2音上昇シンセ / bust: ビットクラッシュ風ノイズバースト
- ミュート永続化・遅延初期化は現行のまま。

## 5. アクセシビリティ・互換

- `prefers-reduced-motion`: 走査線・グリッチ・呼吸発光・カウントアップを停止/即値。
- コントラスト: シアン on `#0a0a12` は十分。ボタン文字は黒 on ネオングラデ。
- ライトテーマ: 既存 `.stage-dark`（テーマ変数固定）がそのまま効く。追加対応不要。
- `aria-label="マス N"` 不変 → 既存 E2E スクリプトがそのまま通る。
