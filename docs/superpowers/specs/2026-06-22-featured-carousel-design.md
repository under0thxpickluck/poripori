# 注目マーケット カルーセル（本家型ヒーロー）設計

**日付:** 2026-06-22
**対象:** 一覧トップに本家Polymarket風の「左大スライド＋右小カード」ヒーローを実装

## 目的

マーケット一覧の最上部に、注目マーケットを左の大きな自動スライド＋右の小カード群で見せるヒーロー領域を追加する。既存の「注目のマーケット（出来高トップ3カード）」ブロックを置き換える。

## コンポーネント

### `src/components/FeaturedCarousel.tsx`（新規）
- props: `{ markets: Market[] }` — 出来高順に選出済みの注目リスト（最大7件想定）
- 振り分け: 先頭3件 = 左の大スライド（自動巡回）、続く4件 = 右の小カード（固定）
- 件数が少ない場合は縮退（スライドは最低1件、右カードは残りで埋める。0件ならコンポーネント自体を描画しない）

#### 左の大スライド
- 大きめヘッダー画像（`MarketImage` 再利用）＋カテゴリチップ＋質問（大きめ）＋YES/NO％バー＋出来高
- カード全体クリックで `/market/:id` へ
- 左右 `◀ ▶` 矢印オーバーレイ、下部にドット
- 自動スライド5秒。ホバー中は一時停止。矢印/ドット操作でも巡回し、手動操作でタイマーをリセット。末尾→先頭ループ
- 価格は `marketPrice(market)` を使用（`MarketCard` と同じ）

#### 右の小カード（4枚）
- サムネ（`MarketImage`）＋質問2行（`line-clamp-2`）＋YES％
- クリックで `/market/:id` へ

### レイアウト
- `lg:grid-cols-3`：左スライド `lg:col-span-2`、右カラム `lg:col-span-1` に小カード4枚を縦積み（`flex flex-col` で高さを左に合わせる）
- lg未満：スライド全幅 → 下に小カードを `grid-cols-2`

## MarketList への組み込み
- 既存の「注目のマーケット」3カードブロックを `<FeaturedCarousel markets={featured} />` に置換
- 表示条件は現状どおり `isDefaultView`（カテゴリ=All・受付中・未検索）のときのみ
- `featured` の選出を出来高順トップ**7件**に拡張（現状は3件）
- 注目7件は下の「すべてのマーケット」グリッドから除外（`rest` で除外済みロジックを流用）

## 非対象
- store / 型 / 管理UI の変更なし（出来高順の自動選出のみ。featuredフラグは導入しない）
- 検証は `npm run build`（tsc + vite）成功をゲートとする
